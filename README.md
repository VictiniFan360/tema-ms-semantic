# MS Semantic WP

**MS Semantic WP** es un tema de WordPress semántico, accesible y personalizable. Es mi tema de WordPress para el proyecto TomCat.

Está construido siguiendo buenas prácticas de desarrollo, evitando estilos inline, priorizando accesibilidad (WCAG) y usando el Personalizador nativo de WordPress.

---

## Características

- HTML semántico
- Enlace “Saltar al contenido principal” accesible
- TopBar configurable desde el Customizer
- Menú de navegación horizontal
- Soporte para logo personalizado
- Colores de Header, Footer y TopBar configurables
- Imagen de fondo configurable por sección
- Tipografías configurables
- Soporte para widgets en el Footer
- Compatible con Bootstrap 5
- Buen contraste y validación WAVE
- Sin estilos inline (CSS limpio)
